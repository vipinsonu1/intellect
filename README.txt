Intellect National Academy - Static multi-page website

Files:
- index.html
- about.html
- programs.html
- admissions.html
- gallery.html
- contact.html
- assets/ (logo.png, favicon.png, admission_form.pdf)

Deploy:
1) Push this folder to GitHub
2) In Netlify: New site -> Import from GitHub -> select repo -> Deploy

Netlify forms are enabled with data-netlify="true" on forms; configure notifications in Netlify site settings.
